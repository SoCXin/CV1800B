### 关键特性

* 1GHz + 700MHz RISC-V C906
* DDR2 64MB
* CVITEK TPU
* 支持H.264/H.265视频编码，最高可达2880x1620@20fps
* 5x UART
* 1个以太网PHY
* QFN68(7mm x 7mm)
